# cybersecurity-threat-detection-
# 🔐 Cybersecurity Threat Detection using Machine Learning

MSc Final Project | University of Roehampton | Data Science

## 📌 Overview
This project detects cybersecurity threats using the UNSW-NB15 dataset.
Models compared: Logistic Regression, Random Forest, SVM, and XGBoost.

## 📊 Results
Best model: XGBoost | See `/Output/model_comparison.png`

## 📁 Dataset
UNSW-NB15 dataset. Large raw files not included due to GitHub size limits.  
Download from: https://research.unsw.edu.au/projects/unsw-nb15-dataset

## 🛠️ How to Run
1. Download the dataset from the link above
2. Open `cybersecurity_threat_detection.ipynb` in Google Colab or Jupyter
3. Run all cells
